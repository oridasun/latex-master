Preguntas Frecuentes (FAQ)

¿Qué es LaTeX Master?

LaTeX Master es una aplicación web gratuita que convierte o elimina el código LaTeX incrustado en textos. Es especialmente útil cuando recibes documentos académicos, científicos o matemáticos con fórmulas LaTeX y necesitas un texto limpio y legible sin necesidad de instalar ningún programa.

¿Necesito instalar algo para usarla?

No. LaTeX Master funciona directamente en el navegador, sin instalaciones, sin registro y sin ninguna dependencia externa. Solo necesitas acceso a internet para cargar el motor matemático (MathJax) cuando usas el modo Renderizar.

¿Qué modos de conversión están disponibles?

La aplicación ofrece tres modos:

  • Renderizar como imágenes: convierte cada fórmula LaTeX en una imagen vectorial (SVG) de alta calidad, lista para pegar en Word, Google Docs, Notion, etc.
  • Unicode: sustituye el código LaTeX por caracteres Unicode equivalentes (por ejemplo, \alpha → α, \frac{1}{2} → ½).
  • Texto plano: convierte las fórmulas a texto ASCII legible (por ejemplo, \sqrt{x} → sqrt(x)).

¿Cómo uso el modo "Renderizar como imágenes"?

Selecciona el modo Renderizar como imágenes, pega tu texto con fórmulas LaTeX en el cuadro de entrada y pulsa Limpiar texto (o Ctrl+Enter). La primera vez puede tardar unos segundos en cargar el motor matemático. El resultado mostrará el texto con las fórmulas reemplazadas por imágenes SVG que puedes copiar o descargar.

¿Puedo personalizar el aspecto de las fórmulas renderizadas?

Sí. En el modo Renderizar aparece un panel de estilo donde puedes ajustar el tamaño (10–48 px), el color del texto matemático, activar negrita y elegir entre fondo blanco o transparente.

¿Qué tipos de código LaTeX soporta la conversión?

La aplicación reconoce los delimitadores habituales ($...$, $$...$$, \(...\), \[...\]) y convierte más de 100 comandos, entre ellos:

  • Fracciones: \frac{a}{b}
  • Raíces: \sqrt{x}, \sqrt[n]{x}
  • Letras griegas: \alpha, \Sigma, \omega…
  • Operadores: \times, \leq, \neq, \approx…
  • Cálculo: \sum, \int, \partial, \infty…
  • Flechas: \rightarrow, \Rightarrow…
  • Super y subíndices: x^2, x_n

¿Por qué algunas fórmulas aparecen resaltadas en amarillo?

Cuando el modo Renderizar no puede procesar una fórmula (por ejemplo, porque usa sintaxis LaTeX muy avanzada o poco común), la muestra resaltada en amarillo con el código original para que sepas que no se pudo convertir. En ese caso, prueba el modo Unicode o Texto plano.

¿Puedo descargar el resultado?

Sí. Una vez convertido el texto, puedes:

  • Copiar el resultado al portapapeles con un solo clic.
  • Descargar HTML para obtener un archivo que conserva el formato e imágenes.
  • Descargar PDF para generar un documento listo para imprimir o compartir.

¿La aplicación guarda o envía mis textos a algún servidor?

No. Todo el procesamiento ocurre localmente en tu navegador. Ningún texto que introduzcas se envía a ningún servidor externo. Consulta nuestra Política de Privacidad para más detalles.

¿Funciona en dispositivos móviles?

Sí. La interfaz es totalmente responsiva y funciona en smartphones y tablets. Para una experiencia óptima con fórmulas complejas se recomienda usar un navegador actualizado (Chrome, Firefox, Edge o Safari).

¿Cómo cambio el idioma de la interfaz?

Pulsa el botón EN / ES en la esquina superior derecha del encabezado. La aplicación detecta automáticamente el idioma de tu navegador al cargar, pero puedes cambiarlo en cualquier momento.

¿Existe un atajo de teclado para convertir rápido?

Sí. Mientras el cursor esté en el cuadro de texto, pulsa Ctrl+Enter (Windows/Linux) o Cmd+Enter (Mac) para lanzar la conversión sin necesidad de hacer clic en el botón.

¿La aplicación es gratuita?

Sí, LaTeX Master es completamente gratuita y sin publicidad. Si te resulta útil, puedes apoyar su desarrollo invitándome a un café ☕.
